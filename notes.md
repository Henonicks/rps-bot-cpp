# Run Locally

If you are on a Linux distro, run the following commands for prerequisites.

This will download the latest version of D++, a C++ library for Discord.

```sh
apt install wget
wget -O dpp.deb https://dl.dpp.dev/
dpkg -i dpp.deb
```

had to install:

> opus-tools
> zlib1g-dev
> libssl-dev
